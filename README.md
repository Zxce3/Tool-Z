## INSTALL ON TERMUX
```php
$ pkg up
$ pkg i ruby git ffmpeg tree zip unzip
$ git clone https://github.com/Zxce3/Tool-Z
$ cd Tool-Z
$ ruby main.rb
``` 

## INSTALL ON LINUX
```php
$ sudo apt-get update
$ sudo apt-get install git ffmpeg ruby tree zip unzip
$ cd Tool-Z
$ ruby main.rb
```
